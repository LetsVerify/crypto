# Intro

> cargo add bbs --git https://github.com/LetsVerify/BBS_rust

> and use:

```rust
use bbs::bbs_bn254::*;
```

> to import


## See usage in [tests](./bbs/tests/)

## Docs

> run `cargo doc --no-deos --open` to generate and open the documentation.

## Benchmarks

> run `cargo bench -p bbs` to run the benchmarks.

## Npm Package
